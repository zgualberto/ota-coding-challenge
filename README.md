

## Installation

```bash
$ npm install
```

Create the sqlite database file on application root folder named `ota.sqlite`

## Running the app

```bash
# run the database migration
$ npm run ota:migration:run

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
$ npm run test .
```
